Description

PerformIndexOperations has been superseded by GroupDetectors, and all usage of it should now be removed, and the algorithm itself marked as deprecated.

Goals:

  • Remove all calls of PerformIndexOperations
  • Mark PerformIndexOperations as deprecated
  • Make it very clear in documentation that the commands relate to workspace indices, not spectrum numbers
