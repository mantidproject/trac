Reported by: Nick Draper Owned by: Nick Draper
Priority: major Milestone: Release 2.1
Component: MantidPlot Keywords:
Cc: Blocked By:

Description

I think this is a problem with qtiplot and not Mantid but still would be brilliant if we could fix it: This happens When several plot windows are on the screen and when using the zoom. You can zoom in and out in one of the plot window, but if you select another plot window, the zoom button is still active but you can not use it in this new window. One needs to click another button (say select button) then click the zoom button before being able to user the zoom again.
