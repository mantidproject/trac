Note: The first use dialog now only hides itself until the next major or patch release

To Test

  1. Start up mantidplot
  2. You should get the first use dialog (this is the same as first ever use).
  3. Play with the dialog -> for a development version the release notes should be the last major release.
  4. Check do not show me this again
  5. close the dialog using SET
  6. Open the dialog from the Help menu
  7. the do not use checkbox should remember your setting.
  8. Close Mantidplot
  9. start it again, you should not get the first use dialog

when you open it from the help menu it should remember your do not show setting.
