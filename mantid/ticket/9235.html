LoadSassena: add default option to sort by momentum transfer

Reported by: Jose Borreguero Owned by: Jose Borreguero
Priority: major Milestone: Release 3.2
Component: Indirect Inelastic Keywords:
Cc: lynchve@… Blocked By:
Blocking: Tester: Vickie Lynch

Description (last modified by Jose Borreguero) (diff)

It is not uncommon to find that the order of the structure factors is not according to increasing Q-modulus, which will lead to analysis errors if unaware or forget about this fact. Thus, it is highly desirable to add a default option to sort the data as is loaded.

TESTER: download and uncompress attached file test.tar.gz under directory /tmp, then go to /tmp/test and execute script test.py in MantidPlot script window.

Group workspaces "x" and "y" should be created, the first with unsorted momentum transfer, the second one with sorted values. Compare workspaces x_qvectors and y_qvectors, as well as the order of the histograms in the other workspaces.
