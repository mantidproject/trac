How to verify this ticket

  1. Load VULCAN run 20002;
  2. Execute algorithm CreateLogTimeCorrection() to generate a correction table;
  3. Execute GenerateEventFilter() to generate a filter according to log "Special1_0";
  4. Execute FilterEvents() with and without the correction table. The outcome should be close but not exactly the same.

Download the attachment "verify6844.py" which contains the script to verify this ticket.
