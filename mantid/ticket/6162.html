Ticket #6162 (closed: fixed)

Opened 8 years ago

Last modified 5 years ago

Remove Peak via the InstrumentView

Reported by: Owen Arnold Owned by: Roman Tolchenov
Priority: blocker Milestone: Release 2.4
Component: Mantid Keywords:
Cc: nick.draper@… Blocked By:
Blocking: Tester: Owen Arnold

Description

The InstrumentView has some nice functionality for overlaying a peaks workspace and viewing the peak positions on the instrument plot.

Pascal has been asking (for quite some time) that we allow users to remove a peak from the peaks workspace via the instrument view. Change made in the instrument view should be reflected in the peaks workspace.

We should probably enable a right-click menu for removing the selected peak. I strongly suggest enlisting Nick Drapers help in determining how this should work before any coding is done. This functionality should also be trialled by Pascal before the ticket is marked as complete.
