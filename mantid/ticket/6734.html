NaCl SXD Reduction to System Test

Reported by: Owen Arnold Owned by: Owen Arnold
Priority: major Milestone: Release 2.5
Component: Mantid Keywords:
Cc: Blocked By:
Blocking: Tester: Michael Reuter

Description

Dennis and I worked to produce a reduction script, which would accurately find the lattice parameters of a NaCl sample run on the ISIS SXD instrument. This appears to be producing some nice results. As this is a good benchmark for future work in this area we ought to integrate the results into our system test framework. Having this as part of our system tests, will help build confidence in our single crystal elastic diffraction setups.

I'm going to try to avoid making the system tests too sensitive. Measuring the lattice parameters to 1 decimal place is probably a sufficient test that things are working okay.
