Add a preference option to draw all error bars on new graphs.

Reported by: Roman Tolchenov Owned by: Roman Tolchenov
Priority: major Milestone: Release 3.2
Component: GUI Keywords:
Cc: Blocked By:
Blocking: Tester: Karl Palmen

Description

When a new graph is plotted with error bars by default not all error bars may be shown to make the graph clearer. Some users prefer to see all error bars in any case.

Add an option to the Preferences dialog to toggle the ways of drawing error bars on newly open graphs.
