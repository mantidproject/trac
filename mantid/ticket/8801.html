To Tester:

This ticket adds functionality to DensityOfStates to load partial DOS data from file. The additional (optional) ions parameter takes a list of elements and loads them from file (if they're present) and output them as a group workspace with one workspace for the contributions of each ion.

The new summed contributions option will (unsurprisingly) produce a single workspace with the sum of each of the individual contributions.

Check that both of these options work as you'd expect. There are a couple of system tests in DOSTest.py that you can use as a reference for running the program and example output.
