Description

We have #9367 (currently a work in progress) to deal with the IDF problems within the SANS ISIS interface. This is not a trivial job, and so in the meantime, just so the scientists can use the interface, it has been suggested we add a "SANS2DTUBES" option to the instrument dropdown menu as well as the ISISCommandInterface as a way to specify the latest version of the SANS2D instrument.

While I am changing the drop down, I will also remove the SANS2D_EVENT option.
