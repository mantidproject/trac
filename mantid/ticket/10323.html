VESUVIO diffraction

Reported by: Dan Nixon Owned by: Dan Nixon
Priority: major Milestone: Release 3.3
Component: Framework Keywords:
Cc: Blocked By: #10215
Blocking: #10278 Tester: Harry Jeffery

Description

There are issues with using the current Indirect Diffraction interface with VESUVIO:

  • The parameter file does not have the required workflow information about the monitors (also fixed by loading using LoadVesuvio with a PAR file)
  • MSGDiffractionReduction only outputs a single workspace with a single spectra, Andrew would like it to output a workspace for each spectra

The easiest way to ix this is likely to create a separate diffraction algorithm for VESUVIO and integrate it into the Indirect Diffraction UI (adding a field for the PAR file).

To test:

  • Open Indirect > Diffraction
  • Select VESUVIO
  • Load a sample file (e.g. 17297 from the data archive or 15289 from system tests)
  • Select plot type Spectra and click Run
  • Now enable Use Individual Grouping and select plot type Contour and click Run
  • (This does not perform grouping and leaves each detector in its own spectrum, which is used by instrument scientists to verify the detector positions)
