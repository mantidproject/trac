In order to test this ticket you must not have access to the archives

To test

  1. Log into the catalog using your federal ID.
  2. Enable debugging in the console log.
  3. Perform a search.
  4. Select an investigation.
  5. Select a datafile to download (or several) and click "Download to...". The files should be saved to that directory.
  6. Select "Load" for a different datafile and the file should be downloaded and loaded into a workspace.

As you have debug mode enabled each time you download a datafile a URL is generated. You can use this URL to verify (via a browser) that the file you downloaded is the same as the one you selected.
