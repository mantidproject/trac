To test:

  • Open Indirect > Data Analysis > Elwin
  • Load a single file (_red or _sqw, from either IRIS or OSIRIS) form systemtests
  • Enable Plot and Normalise to Lowest Temp
  • Run
  • The ELF workspace will contain a series of single points with the same X value over several spectra
  • The ELT workspace will contain several points with the same X value and same Y value of 1 (you would never actually use this option when running Elwin with a single file)
  • The Q and Q2 workspaces will just look like normal ElasticWindow workspaces
