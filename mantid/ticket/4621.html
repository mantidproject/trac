Add option to output in cm-1 to CTE (TOSCA) interface.

Reported by: Peter Parker Owned by: Peter Parker
Priority: critical Milestone: Release 2.0
Component: Mantid Keywords:
Cc: Blocked By:
Blocking: Tester: Andrei Savici

Description

Alternative, temporary implementation of ticket #4512, to get at least some of this functionality in before the upcoming release:

Add a property to the Tosca parameters file, along the lines of "present user with convert to cm-1 units choice, within the Convert to Energy interface?". Refer to this property when the interface starts up, and show a checkbox in load options. When checked, the contents of the loaded workspace will have been converted to cm-1. (Essentially just multiply the x axis by 8.065.)
