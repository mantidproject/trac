Allow PlotPeakByLogValue to take a minimizer string

Reported by: Dan Nixon Owned by: Dan Nixon
Priority: major Milestone: Release 3.5
Component: Framework Keywords:
Cc: Blocked By:
Blocking: #11561 Tester: Harry Jeffery

Description (last modified by Dan Nixon) (diff)

Allow this to accept a string like Fit to configure a minimizer with options (e.g. FABADA).

Also add a MaxIterations property with the value passed directly to Fit.

There will be the option to make the minimizer string templated to include workspace information (i.e. name and spectrum index) so that output is not overwritten on minimizers such as FABADA that output workspaces.
