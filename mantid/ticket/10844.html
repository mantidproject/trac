Indirect: Calibration tab does not run on OS X/Linux from a package

Reported by: Martyn Gigg Owned by: Martyn Gigg
Priority: critical Milestone: Release 3.3
Component: Indirect Inelastic Keywords:
Cc: Blocked By:
Blocking: Tester: Owen Arnold

Description

The reducer used by the inelastic workflow algorithm, InelasticIndirectReduction, requires that the output path be specified else it will simply use the working directory of the application. This directory is not writable on Linux/OSX when run from an application shortcut.

This prevents the inelastic GUI from being used on these platforms.
