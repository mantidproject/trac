To test:

  • Non frame-skipping mode: Start the ORNL SANS UI and reduce run 1466 from the system tests data directory. For the transmission, you can put the same run for the scattering data file, the sample direct beam file and the empty direct beam file. The transmission output will be 1.0. After the reduction, two transmission files should be written to disk: one for the transmission data, and one for the fit to that data. The files should appear in the directory that you chose on the first tab of the reduction UI as your output directory.
  • Frame-skipping mode: Do the same with run 3293 from the system tests data directory. Two transmission files should also be written to disk.

Frame-skipping mode and non-frame-skipping mode will both produce two transmission files, but the processing goes through a different path in the code. Performing both tests ensures that the save method is correctly called in both paths.
