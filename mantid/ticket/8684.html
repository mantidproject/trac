ISIS Reflectometry Guide

Reported by: Owen Arnold Owned by: Owen Arnold
Priority: major Milestone: Release 3.1
Component: Reflectometry Keywords:
Cc: Blocked By:
Blocking: Tester: Arturs Bekasovs

Description

There has been a significant amount of work performed during this iteration on various aspects of the back-end scripting and Algorithms for reflectometry reduction at ISIS. It would be extremely useful to create a guide to these features so that instrument scientists and new users can navigate their way through them.

The reflectometry work is not complete, and the wiki page should indicate this. We should also link to it from the reflectometry gui via the help, and ensure that Max Skoda, who can update this page as well.

This page would also be a useful feature for the release notes.

The guide is well-written and exhaustively describes what tools are available and what is their current status. Code snippets run to completion, resulting plots do seem OK (though my lack of experience with reflectometry doesn't allow me to verify that they are correct).

The only problem I've found is not with the guide itself, but rather with a few pages linked from it: http://www.mantidproject.org/ReflectometryReductionOne, http://www.mantidproject.org/PerformIndexOperations and http://www.mantidproject.org/CreateTransmissionWorkspace.

These are all auto-generated and because the wiki-formatted description of the algorithms has leading space on all the lines, it is treated in a special way, which leads to these unnecessary borders to be created around the paragraphs. I leave it with Owen to consider whether it's worth fixing.
