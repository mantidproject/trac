[C2E] - Energy Transfer Tab Additional Feature - (Scaling)

Description

Convert to Energy interface:

In the Energy Transfer tab, give users the option to scale the input workspace by the value in an input box; this will be done via the algorithm Scale.

To test:

  • Select an instrument in the C2E GUI. E.g. TOSCA.
  • Select a corresponding .raw file. E.g. TSC10076.raw (from the AutoTestData folder).
  • Click 'Run Energy Transfer'.
  • Plot the spectrum of the workspace which is generated.
  • Return to the 'Energy Transfer' tab.
  • Enter a new value in the 'Scale: Multiply by' input box.
  • Click 'Run Energy Transfer'.

There should be a visible change in the scale; the entire workspace is multiplied by the value entered.
