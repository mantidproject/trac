To test:

  1. Open ISIS SANS GUI
  1. pick some mask file
  1. Load e.g. SANS2D run 14256 (but any sans2d run will do, although some sans2d runs have no front detector data, i.e. all zeros)
  1. Go to analysis details tab. For 'Wavelength' combobox select either Ranges Linear or Ranges Log.
  1. Specify a range string. E.g. '2.5, 5.5' or '2.5, 5.5, 9.5'
  1. Select from the Detector bank combobox either 'both' or 'merge'
  1. Hit 1D reduce button after hitting the Load Data button

What you should see is at least the generation of both 'rear' and 'front' workspaces for the selected ranges
