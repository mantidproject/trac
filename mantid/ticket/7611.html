comment:2 Changed 7 years ago by Arturs Bekasovs

Results of the last meeting:

Scientists agreed on the suggestion to keep plots in the interface and make it possible to export them to the workspaces in the ADS.

Feature request: make it possible to set last_run to "Auto" which will make the interface to observe changes in the directory of the first_run and append any new files to the data set. Is a great enhancement and will need to look into that as soon as basic functionality is working. (#9214)

Addition: need to add missing parameters from the AsymmetryByLogValue, such as integration start_x and end_x and integration type - integral, differential. (Was added in the new version of the mock-up)

Feature request: on the loading step, make it possible to play with parameters without reloading the files. This means keeping all the original data in memory, which might not be feasible. Wait for Stephen or Jamie to provide a large data set to play with it. (#9215)

Addition: in addition to specifying data sections for polynomial function, specify some points which will be used for fit instead of the original data. This might be done after section-based approach is implemented, and used as another "mode". Anders suggested a spline fit function could be used for that. (#9216)
