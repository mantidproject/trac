Description

From Steven:

Fitting a run and then opening the sequential fit dialog automatically leaves the first run in the dialog. This is great, but at the moment the entry is neither a pure number nor the full filename, i.e. fitting run 45510 on EMU leaves 'EMU45510' in the dialog. I think it better to just have '45510' (the filename would be 'EMU00045510.nxs')
