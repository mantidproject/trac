comment:1 Changed 11 years ago by Russell Taylor

Francis Pratt had this problem and solved it by updating his graphics card drivers:

OK, the graphics card on my desktop is an ATI FireGL V3100,

Original driver version 8.563.0.0, dated 10/12/2008 Installed an updated driver from the ATI website, version 8.583.0.0, dated 11/02/2009 seems to fix the problem

My laptop had the same window freezing problem with the mobile version of the same card, ATI Mobility FireGL V3100 updating the driver from the Dell website to version 8.353.1.1000, dated 8/5/2007 also fixed the problem

Francis
