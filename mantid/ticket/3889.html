Remove option "vulcancorrection" in Instrument::calcConversion

Reported by: Wenduo Zhou Owned by: Wenduo Zhou
Priority: major Milestone: Release 2.0
Component: Mantid Keywords:
Cc: Peter, Peterson Blocked By:
Blocking: Tester: Vickie Lynch

Description

As the calibration file of VULCAN can be generated as the same definition of other powder diffractomers' calibration file, there is no more need to have the special way to calculate conversion (i.e., 1/DIFC*(1+Offset)) for Vulcan. Thus, remove the option "vulcancorrection", and correct the programs using Instrument::calcConversion()
