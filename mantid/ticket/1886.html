TOF correction for diffraction event data

Reported by: Vickie Lynch Owned by: Wenduo Zhou
Priority: major Milestone: Backlog
Component: Framework Keywords:
Cc: Blocked By:
Blocking: Tester:

Description

New algorithm for Mantid that corrects the event data for moderation time delay incurred by neutrons as a function of wavelength. Implement same corrections that are in Jason Hodges's Fortran code. Then focussing should become a simple linear relationship for all pixels.
