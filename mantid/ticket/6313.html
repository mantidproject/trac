[IDA] Modify ElWin, msdFit and FuryFit procedures in IDA

Reported by: Dereck Kachere Owned by: Dereck Kachere
Priority: major Milestone: Release 2.4
Component: Mantid Keywords:
Cc: Blocked By:
Blocking: Tester: Stuart Campbell

Description

In procedure ElWin, instead of creating a workspace/file of eq1 & eq2 for each individual run, a single ws/file is created with a histogram for each run. The histograms are labelled with the run number as text. eq2 is now log(intensity) instead of intensity. Procedure msdFit is therefore also modified to take these changes.

In procedure FuryFit, the workspace/file of the fitted parameters now includes the background (labelled A0).
