Improve handling of vertical axis in Matrix window

Reported by: Dan Nixon Owned by: Dan Nixon
Priority: major Milestone: Release 3.4
Component: GUI Keywords:

Description

Currently if you view the data for a workspace that has a BinEdgeAxis as the vertical/spectrum axis the values you will see are the lower bin edge, this leads to confusion as you would expect this to be the bin centre given that this is what is shown for the X axis.

A good demonstration of this is irs26176_graphite002_sqw.nxs.
