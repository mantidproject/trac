Empty ConvertToMD PreprocDetectorsWS property is ambiguous

Reported by: Alex Buts Owned by: Alex Buts
Priority: major Milestone: Release 2.3
Component: VATES Keywords:
Cc: Blocked By:
Blocking: Tester: Andrei Savici

Description

Empty PreprocDetectorsWS property supposes 1) to disable preprocessing detectors positions using the same workspace for sequence of calculations over workspaces and 2) force preprocessing detectors positions for each calculation.

While working from Python, empty property set from GUI forces default property value, and the default is the calculating positions using internal workspace.

Other property value (e.g. "-") is needed to facilitate 1)&2)
