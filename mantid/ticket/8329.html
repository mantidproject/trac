Color fill plot display in the tiled display

Reported by: Owen Arnold Owned by: Roman Tolchenov
Priority: critical Milestone: Release 3.4
Component: GUI Keywords:
Cc: Blocked By:
Blocking: Tester: Dan Nixon

Description (last modified by Roman Tolchenov) (diff)

This is related to #8328. Once that ticket has been fixed. The users want the color fill plots to appear in a tiled fashion on the screen. This is particularly useful if they have 20+ workspaces, as the reflectometry users currently do.

This would involve setting the windows position and sizes according to the number of workspaces being plotted.

Following a discussion with Nick and Owen it's been decided that display of multiple color fill plots will start with an overview multi-layered plot. The multi-layer functionality needs to be extended to allow scrolling the overview graph and picking individual layers to make them separate floating windows.
