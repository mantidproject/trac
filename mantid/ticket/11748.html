Warn when saving nexus files with no instrument XML

Reported by: Harry Jeffery Owned by: Harry Jeffery
Priority: critical Milestone: Release 3.4
Component: Framework Keywords:
Cc: Blocked By:
Blocking: Tester: Dan Nixon

Description

If a Nexus file is saved from a workspace whose instrument has no XML, then Mantid is unable to re-open it. This notably happens after saving a nexus file derived from an .nxspe file, or one created with CreateSampleWorkspace.

We should give a strong warning when users attempt to save such files.
