comment:14 Changed 7 years ago by Dereck Kachere

  • Status changed from accepted to verify
  • Resolution set to fixed

Tester:

  • Open up the Calculate Corrections tab in IDA.
  • Leave Sample Shape as 'Flat'

For each of the attached files and enter the following values:

  • Thickness = 0.5
  • Can Angle to Beam = 0.001
  • Beam Width = 3.0
  • Number Density = 30
  • Scattering cross-section = 2.0
  • Absorption cross-section = 8.0

Select 'Both' in the Plot Output drop-down menu, then 'Run'.

Compare the information in the log when you check and uncheck the 'Verbose' option; more should appear when it is checked.

Do a similar comparison in the Apply Corrections tab. Input the attached files separately in the input boxes and select 'Use Corrections' and 'Use Container' - this uses the files generated in Calculate Corrections so be sure to manage directories.
