comment:10 Changed 6 years ago by Wenduo Zhou

It works for me. Try this setup:

  • Instrument: PG3
  • Run Number: 4844
  • Calibration file: PG3_FERNS_d4832_2011_08_24.cal
  • Characterization file: PG3_characterization_2011_08_31-HR.txt
  • Disable 'Empty Run Correction', 'Vanadium Run Correction' and 'Vanadium Background Run Correction'

In filter setup:

  • Filter start time: 275
  • Filter stop time: 5220
  • Run Number 4844
  • Log Name: SampleTemp
  • Minimum Log Value: 299.9
  • Maximum Log Value: 300.1
  • Log step size: 0.1

You should see that there are 3 splitted workspaces, which are of same number of workspaces shown in splitinfo table.
