comment:10 Changed 7 years ago by Arturs Bekasovs

Tester:

That's a very simple one.

  1. Load AutoTestData/MUSR00015190.nxs. Make sure that period widgets are updated properly - there should be 2 periods overall, and the second period box should be enabled and set to None.
  2. Set period algebra used to 1 + 2.
  3. Load AutoTestData/MUSR00015191.nxs. As it has the same instrument AND number of periods, algebra should remain 1 + 2. You can check that this algebra is actually used by plotting counts of any of the groups - counts will be approx. twice as big as if you use 1 + None.
  4. Load AutoTestData/MUSR00022725.nxs. It has one period only, so widgets SHOULD get updated to reflect it.
  5. Again, load AutoTestData/MUSR00015191.nxs and set 1 + 2.
  6. Now load AutoTestData/argus0031800.nxs. It also has 2 periods BUT the instrument is different, so period algebra should get reset to something like 1 + None.
