1 - This file depends on files that you will find inside ticket #8395. You will need to create the LARMOR files, and download the maskfile and the direct correction file.

2 - After the step 1, you will be able to run the larmor_reduction example. Just open it inside Mantid and run the entire file.

3 - Now, you should read the entire file and check that you can understand how the reduction is performed inside SANS instruments. I mean, what are the steps, what are the inputs.
