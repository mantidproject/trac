Component: Diffraction Keywords:
Cc: Blocked By:
Blocking: Tester: Owen Arnold

Description

Sometimes experimental data from POLDI runs can be merged. The count data from several hdf-files are simply added, which can be achieved using the Plus-algorithm. But before, some sanity-checks have to be performed, since it is only allowed to merge data from experiments with the exact same parameters (sample table position, chopper speed, ...).
