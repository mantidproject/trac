Ticket #1972 (closed: fixed)

Opened 10 years ago

Last modified 5 years ago

Change skip plotting error bars to be unnoticeable.

Reported by: Stuart Campbell Owned by: Roman Tolchenov
Priority: major Milestone: Iteration 27
Component: MantidPlot Keywords:
Cc: Blocked By:
Blocking: Tester: Brett Tully

Description

The plotting in MantidPlot will intentionally skip drawing some error bars. For some data this is quite noticeable and means that the plots cannot be used for publications etc...

I think that we need to either alter the threshold of this plotting, and/or provide a means to disable it.

Or maybe, we should take the approach Origin has, which is to draw errors for all the points that are drawn, but for large data sets, only plot every N points.
