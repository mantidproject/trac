Instruction to verify

  1. Download all the attached files;
  2. Build Mantid before merging this ticket's branch to master.
  3. Run script verify_7704.py.
  4. In the output workspace PG3_11486_Guessed, plot spectrum "Data" (i.e., raw experimental data) and "InBkgd" (background calculated from input parameters). You can see the background is way off;
  5. Merge the branch to master and build Mantid.
  6. Re-run the script, and you can find that the background match the data.
