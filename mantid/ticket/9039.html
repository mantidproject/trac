Description

The following is a reasonable request by the users that we should get done 

Somewhat minor, but a nuisance: the list of runs on the left does
not scale to the maximum width of the titles, so it pops in an
annoying scroll bar. Ideally we'd like to be able to resize the box
once the list has been read in to the longest entry (sizehint), and be
able to rescale it manually if required.
