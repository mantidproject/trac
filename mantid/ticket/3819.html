This ticket has been resolved in ticket #7639. However, I was unable to suppress the general error message created by the algorithm class (Error in execution of algorithm CatalogLogin:), and have added a more helpful and user friendly message.

To verify this is true:

  1. Open Mantid
  2. Goto Catalog -> Login
  3. Input invalid username, password, or both (same error message returned for all).
  4. Error message returned: Username or password supplied is invalid.

Similar functionality has been implemented for ICat4 and for the logout routine in ICat3 and 4. To test this view the comments on #7639, specifcally ​comment:25. To code review see ​comment:30 and ​31
