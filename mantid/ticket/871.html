GUI to enter sample chemical formula and calculate input to CylinderAbsorption

Reported by: Russell Taylor Owned by: Michael Reuter
Priority: minor Milestone: Release 2.6
Component: Framework Keywords:
Cc: Blocked By:
Blocking: Tester: Nick Draper

Description

This has been requested by the Crystallography group (in particular GEM/HRPD). It is necessary so that they can calculate the absorption for an individual sample (at the moment they just use it for vanadium, which is always the same input).

Ariel has something like this. You enter the chemical formula of the sample, and the packing fraction or the mass/volume of sample and it calculates the required inputs to the CylinderAbsorption algorithm (attenuation and scattering cross-sections and the number density). Also, Matt Tucker has an Excel spreadsheet which does the same look-up.
