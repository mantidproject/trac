LoadPreNexus

Reported by: Peter Peterson Owned by: Peter Peterson
Priority: major Milestone: Release 2.0
Component: Mantid Keywords:
Cc: Blocked By:
Blocking: Tester: Stuart Campbell

Description

New workflow algorithm that does all of the magic of parsing a runinfo file enough to find out what the names of the event files are and load them. The operations are:

  • LoadEventPrenexus for each neutron_event.dat file
  • Merge them together into a single workspace
  • LoadLogsFromSNSNexus to get the rest of the logs onto the workspace
  • LoadPreNexusMonitors for the monitors

This is because of broken event nexus files on ARCS.
