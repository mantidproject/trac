Description (last modified by Wenduo Zhou) (diff)

Implement and/or enhance algorithms and scripts, which help to calibrate powder diffractometer peak profile parameters.

Here is the list of tasks to consider:

  1. Make ConvertInstrumentFile support more instruments such as NOMAD;
  2. Make ConvertInstrumentFile more robust on reading Fullprof .irf file;
  3. Enhance functionality of ProcessBackground if it fails in real application;
  4. Generate python scripts to calibrate instrument profile parameters more automatically.
