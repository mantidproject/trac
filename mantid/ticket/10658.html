Testing

Note: Before testing, please ensure that the systemtests/Data directory is searched by Mantid as one of its search directories.

  • Open Custom Interfaces → ISIS Reflectometry (Polref)
  • Set the instrument to OFFSPEC
  • Create three rows
    • The Run(s) column should contain 27575, 27576, 27578, with one value in each row, respectively.
    • The angle column should contain 0.35, 1.00, 1.70, with one value in each row, respectively.
    • The transmission run column should contain 27585 on every row.
    • Leave Q min and Q max blank.
    • Set dQ/Q to 0.01 on every row.
    • Set the group column to 0 for every row.
  • Hit process with all three rows selected.
    • Processing should succeed without an error.
  • Plot the IvsQ_27575_27576_27578 workspace. With logarithmic x and y axes it should look like the attached screenshot.
