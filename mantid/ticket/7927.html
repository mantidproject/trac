comment:2 Changed 7 years ago by Roman Tolchenov

To tester:

  • Open IDA Conv Fit tab. Use the attached files (_red for File, _res for Resolution).
  • Select "Two Lorentzians" in Fit type.
  • Checkbox Tie Centres must show up.
  • Leave the checkbox unchecked.
  • Set initial values for the Lorentzians: 1) Ampl: 1.0, Centre: 0, HWHM: 0.01; 2) Ampl: 1.0, Centre: 1.4, HWHM: 0.4
  • Select Plot Output
  • Click Run
  • The fit must be good for both peaks (the big at 0 and the small at 1.4)

When Tie Centres is checked after clicking Run the Lorentzians' centres must be equal, goodness of the fit is unimportant.

When Fit type changes to Zero or One Lorentzian Tie Centres disappears.
