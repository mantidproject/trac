comment:33 Changed 6 years ago by Roman Tolchenov

To tester

  1. Load a workspace with similar looking data (eg a musr set from autotest data). The data may need rebinning as there is no fitting range setting in the interface yet.
  2. Start the interface: Interfaces->General->Multi dataset fitting.
  3. Add a few spectra, not too many (~10)
  4. Define a function. Set some parameters global others leave local.
  5. Run a fit. See that there is correct output.
  6. Try setting different initial values, ties, constraints.
  7. Play with the plots, zooming, panning, etc.
