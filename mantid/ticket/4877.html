comment:25 Changed 7 years ago by Dereck Kachere

Tester:

  • Download the attached files, put them in a file and set that file as destination in 'Manage User Directories'
  • Open up the ApplyCorr tab in 'Interfaces' -> 'Indirect Data Analysis'

Inputs:

  • file = irs26173_graphite002_red.nxs
  • Geometry = Flat
  • Use Corrections & Use Container: tick the box
  • flie2 = irs26176_graphite002_red.nxs

Experiment and compare the rest of the options.

  • Selecting the 'Verbose' option should result in more information being displayed in the logger in MantidPlot (you could compare the outputs in Notepad2).
  • 'Scale' option should change the axes.
  • 'Save Result' should save an output workspace in your directory.
  • 'Plot Contributions' should plot contributions, even if 'Plot Output' is set to 'None'.
  • 'Plot Output' options should result in different graphs being plotted.
