Stepping through sequential fits with plot output

Reported by: Dan Nixon Owned by: Dan Nixon
Priority: major Milestone: Release 3.3
Component: Framework Keywords:
Cc: spencer.howells@… Blocked By:
Blocking: Tester: Harry Jeffery

Description

Currently the only way to inspect each spectrum in a fit workspace is to either do an entire sequential fit or do as many single fits as they are spectra. It would be good to have a way to step through a sequential fit, allowing inspection of each spectrum after it has been fit and continuing when the user chooses to do so.

In a message from Spencer:

Another idea from Vicky.
At present we just plot at the end the fit for spectrum 1.
She likes seeing the fits for each spectrum as it happens.
The miniPlot doesn’t do change of axes or zoom so a possibility is to plot the fit for each spectrum as it happens.
In my Modes version it would step thru it – plotting & pausing at each spectrum.
