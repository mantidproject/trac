Waterfall plots have been added in #3139.

I don't see a strong case for any of the other plot types. Vector plots may possibly be useful to someone, though I don't know how a workspace would map onto them. I suggest we leave it unless specifically requested. Histogram, box & pie plots are actually available from a Table so if anyone really wants to use them (unlikely, I think) there is a way to do it.

The adding of arrows, lines and images to plots is already available. I've created ticket #5291 (with a low priority) for porting in the ability to add shapes (rectangles & ellipses). Not sure it's super-useful, but it looks fairly easy.
