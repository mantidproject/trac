MUON: Add warning that dead times are not implemented in ARGUS files.

Reported by: Robert Whitley Owned by: Robert Whitley
Priority: major Milestone: Release 2.2
Component: Mantid Keywords:
Cc: anders.markvardsen@… Blocked By:
Blocking: Tester: Peter Parker

Description

The way ARGUS files are made is different from how EMU, MuSR, and HiFi are constructed. This made it impossible to apply dead time correction based on what is stored in the file for ARGUS data. This has been the way for a while but I've recently noticed that there is no warning to the user that this is happening and to the untrained eye you could believe that it had been applied.

Include a warning message stating that dead times couldn't be applied to ARGUS files. This will help users who are unfamiliar with Muon Analysis.
