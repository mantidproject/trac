Muon Analysis UI on Mac nigglies

Reported by: Michael Reuter Owned by: Robert Whitley
Priority: minor Milestone: Release 2.0
Component: Mantid Keywords:
Cc: A.J.Markvardsen@… Blocked By:
Blocking: Tester: Michael Reuter

Description

There seems to be some spacing issues with the text boxes in the Instrument section on the Home tab. They come out too short and only show part of the last digit. All of the text boxes on the Plot Options tab suffer from the same problem. There's also an issue with the middle spinner in the second line of the Detector Grouping. It doesn't seem to show the dash nicely like it does in Linux. Also, the Load current run button on the Home tab is too short (Linux & Mac) and doesn't show all the text.
