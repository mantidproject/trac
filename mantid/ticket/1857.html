Description

Why is it that, whereas all the framework dependencies are nicely compiled and waiting for us in Third_Party, we have to go through linux-esque pain to obtain and build the qtiplot dependencies?

I'm thinking particularly of sip, PyQt & QScintilla here, which seem to be impossible to download directly from Riverbank to ORNL (takes hours and then you get an empty zip file).

I appreciate that Qt is huge, so we probably don't want to add that (though the dlls we actually use are not so big).

If we don't want to put the compiled libraries in we could at least add the source.
