Description

Ticket request from Spencer:

Update the default values used in resolution generation to be the following:

  • rebinning - low/high are +/- 10*resolution from IPF rounded to nearest 0.1
  • width is high/100 rounded to 0.001; background - start is 0.9 low, end is 0.8 low.

Also set the resolution mini plot to auto plot if possible.

Parameters used in resolution generation should be added to logs in the resolution file. Namely: Scale &, if true,Scale Factor; Smooth; Background Start & End; Rebinning low, width & high.
