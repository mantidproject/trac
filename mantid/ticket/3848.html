Here are my steps:

1)      Convert the 29 Gb Iron data file to a MDEW event nexus file .

2)      Load resulting *.nxs file into PV. Takes 623 seconds (reporting given by LoadMD code)

3)      Perform axis-aligned rebin in PV using 50 * 50 * 50 * 50 bins. Takes 1365.5 seconds!
