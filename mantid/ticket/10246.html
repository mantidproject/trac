For testing, I would suggest building mantid on a OS X 10.9 machine and verifying that all unit tests pass. I would also verify that these changes don't create any errors or warnings on other platforms. One can also remove "-Wno-deprecated-register" from the CXX_FLAGS and verify that the "'register' storage class specifier is deprecated" warning does not come from any Mantid code.

Boost 1.56 breaks Mantid, so use an earlier version installed by homebrew ("brew switch boost 1.55.0_2")
