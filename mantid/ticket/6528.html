PeaksViewer Preferences

Reported by: Owen Arnold Owned by: Owen Arnold
Priority: major Milestone: Release 2.5
Component: VATES Keywords:
Cc: Blocked By:
Blocking: Tester: Vickie Lynch

Description

There are a couple of preference options which would be very useful on the PeaksViewer.

1) The ability to control the extent of a cross-hairs peaks marker into the viewing plane. There is no 'correct' value for this, although we already make a sensible choice. However, in some workspaces, the peaks markers are extending too far and therfore do not appear to overlap the peaks when the user slices away from the peak.

2) The ability to control the size of the cross-hairs peaks markers on the projection. Again, we try to provide a sensible default for this, but the users may want to adapt it.
