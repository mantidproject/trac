SliceViewer: show multiple lines

Reported by: Janik Zikovsky Owned by: Stuart Campbell
Priority: minor Milestone: Backlog
Component: Framework Keywords:
Cc: scu@… Blocked By:
Blocking: Tester:

Description (last modified by Janik Zikovsky) (diff)

Overlay several line plots in the SliceViewer/LineViewer.

  • This will require adding a LineOverlay object in the SliceViewer object.
    • Make sure to add the LineOverlay objects in the right order - that is, the second LineOverlay object must be a child of the first one in order for mouse events to transfer.
    • Be aware that there is already another LineOverlay object used for the outline in dynamic rebinning. The 2nd line might need to be a child of that one.
  • The LineViewer will need to support showing more than one line too, somehow?
