Users moaning over the installer

Reported by: Anders Markvardsen Owned by: Nick Draper
Priority: minor Milestone: Iteration 28
Component: Mantid Keywords:
Cc: nick.draper@… Blocked By:
Blocking: Tester: Stuart Campbell

Description (last modified by Nick Draper) (diff)

For example Francis and Steve asked if the following improvements where possible to the installer: (both these users have fairly slow machines 4-6 years old)

  1. After hitting the next button on the "End-User License Agreement" screen sometimes a small window with the text "Please wait.. blah blah .. checking your computer or something...", but most confusing is that this small window have a Return button where it is not clear what would happen if you hit it - would it be possible to disable this window popping up? Since it just seem to be confusing and random.
  2. In the 'Custom Setup' window. When attempting to add the 'Includes' there are a number of options for this: "will be installed on local hard drive", "entire feature will be installed on local hard drive" and "feature will be installed when required", which is totally confusing since these options all seems to do the same thing!
  3. Would it be possible to have an overall process bar, since now when you install MantidPlot on a number of occasion you get the impression that MantidPlot has installed only for the status bar to reset and start all over again.
