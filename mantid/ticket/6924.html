To test:

  • Log on the analysis cluster at SNS.
  • Compile Mantid using cmake with -DMAKE_REMOTE_JOBS=TRUE
  • Add "cluster.submission=on" to your Mantid user properties (in ~/.mantid/Mantid.user.properties)
  • Start the EQSANS reduction UI.
  • Load reasonable reduction parameters (enough to execute a reduction).
  • Make sure to pick a location for the output directory (first tab).
  • Click the "Send cluster" button. You should be prompted for your credential. A job should then be submitted to Fermi. The results will appear wherever the output directory your chose is.
  • Check the STDIN files in your home directory for errors.
  • Once the job is completed. Just click the "reduce" button. The output from running on the cluster and running locally should be the same.

Alternate path:

  • Under the Tools menu of the EQSANS reduction UI, pick "Use Mantid Python API v2".
  • Repeat the test above after having selected the API v2.
