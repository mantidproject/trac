SANS Q resolutions for ISIS (SNS)

Reported by: Anders Markvardsen Owned by: Peter Parker
Priority: major Milestone: Release 3.5
Component: SANS Keywords:
Cc: richard.heenan@… Blocked By: #10689, #10691, #10692, #10775, #10987
Blocking: Tester:

Description

Based on detailed investigation #2161.

The main purpose of this ticket is to enable Q resolution to ISIS SANS. As discussed in #2161 a straight copy is not feasible at this point, for example, because of the differences in preferred way ISIS and SNS scientists like to sum contributions for different wavelength into Q.

As a minimum this umbrella ticket will entail:

1) Implementation of new TOFSANSResolutionByPixel algorithm that output the resolution by pixel into a workspace that can a) be reused for more than one sans reduction and b) be plotted for inspection by advanced user

2) Take output of 1) and allow it as input into Q1D for summed Q resolution

The main purpose of this ticket is to enable ISIS Q resolution but where possible to share code with SNS.
