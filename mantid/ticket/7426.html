Instruction to verify

  1. Locate gsas data file gss1.txt in mantid/Test/AutoTestData/
  2. Be noted that there are 3 banks in gss1.txt. They are 1, 3 and 5 respectively;
  3. Run LoadGSS on gss1.txt by setting 'UseBankIDasSpectrumNumber' to true;
  4. Check the generated workspace such that the 3 spectra have spectrum number as 1, 3 and 5. An easy way to find out this is to plot the workspace;
  5. Run SaveGSS on the workspace by setting 'UseSpectrumNumberAsBankID' to true.
  6. Compare the newly generated GSAS file with gss1.txt
