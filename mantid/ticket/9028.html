Add a scaling property to SNSPowderReduction

Reported by: Peter Peterson Owned by: Peter Peterson
Priority: major Milestone: Release 3.2
Component: Diffraction Keywords:
Cc: Blocked By:

Description

POWGEN would like their data multiplied with a number (e.g. 100) before being saved to gsas, fullprof, and topas. This is so when the data is loaded into gsas it will have room on the plot for their tick markers denoting peak location. This is a workaround in mantid to accommodate them.
