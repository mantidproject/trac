Branch is ​here.

To test:

  1. Start MantidPlot and load a TOPAZ run, such as TOPAZ_3132_event.nxs, and convert it to an MD workspace. The quickest way to do this is to bring up the Interface: SCD Event Data Reduction (SEDR), and use the Browse button on the first form to select the event nexus file. Set the Max |Q| to Map to MD to 25, and turn on the Lorentz Correction. Press the Apply button to both load the file and convert it (properly) to an MD workspace.
  2. Switch to the Find Peaks tab and hit Apply
  3. Switch to the Point Info tab and move window aside
  4. Open MDEventWorkspace in VSI and switch to SplatterPlot view
  5. Enter pick mode by clicking the Pick Mode button
  6. Place the mouse over peak and hit the p key
  7. Observe that information is displayed in the SEDR Point Info tab
  8. Exit pick mode by clicking the Pick Mode button
