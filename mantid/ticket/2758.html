Ticket #2758 (closed: fixed)

Allow specifiacation of the other SANS transmission detector.

Reported by: Steve Williams Owned by: Steve Williams
Priority: major Milestone: Iteration 29
Component: Mantid Keywords:
Cc: Blocked By:
Blocking: Tester: Mathieu Doucet

Description

In ISIS SANS runs the transmission is calculated from the number of neutrons reaching the a post-sample monitor (normally detector ID = spectrum = 3) and the pre-sample (normaliation) monitor (normally 2).

The pre-sample monitor is specified in the user file with MON/TRANS/SPECTRUM=? and can be overriden in the monitors section of the Analysis Details tab of the SANS GUI.

Todo: The post-sample monitor will be specfied with TRANS/TRANSPEC=? in the user file and overriden by a box on the GUI similar to above. There will be documentation inhttp://www.mantidproject.org/SANS_User_File_Commands and http://www.mantidproject.org/Using_the_SANS_GUI .
