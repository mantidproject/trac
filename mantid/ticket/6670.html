New version of ModeratorTzero

Reported by: Jose Borreguero Owned by: Jose Borreguero
Priority: major Milestone: Release 2.6
Component: Indirect Inelastic Keywords:
Cc: Blocked By:
Blocking: Tester: Mathieu Doucet

Description (last modified by Nick Draper) (diff)

Current version assumes that the average emission time delay follows a linear relationship with the neutron wavelength. This is correct when the incoming neutron flux features a narrow range of energies. This is the case for indirect instruments that probe quasi-elastic processes.
Other indirect instruments (TOSCA, VISION) use an incoming neutron flux with a wide range of energies. There is no linear relationship between the average emission time and the wavelength of the incoming neutron
The new version of the algoritm will employ an arbitrary relationship between the emission time delay and the energy of the incoming neutron. Thus, it will be applicable to any indirect instrument.
The relationship will be defined in the instrument parameters file. See VISION parameter file for an example, entry "t0_formula".
