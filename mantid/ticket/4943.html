Create Paraview Debian Developer Package

Reported by: Owen Arnold Owned by: Owen Arnold
Priority: major Milestone: Release 2.1
Component: VATES Keywords:
Cc: reuterma.ornl.gov Blocked By:
Blocking: Tester: Michael Reuter

Description

Create a meta-package for the Paraview development dependencies.

The InstallPV.sh script is great, but its a pain to figure out all the dependencies on a clean environment. The Paraview CMake configuration doesn't check or fail on all the dependencies that are actually needed, which can make building an iterative and frustrating process.

This package should make set-up really easy.

For ubuntu 10.04
