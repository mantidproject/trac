Component: VATES Keywords:
Cc: Blocked By:
Blocking: Tester: Michael Reuter

Description (last modified by Owen Arnold) (diff)

Diffraction_Workflow.py in our script repository is a useful script, especially for those getting to grips with the SX reductions. The workspace that it uses is not in our system test data repository, or our autotest repository. This makes it hard to execute. The system test Diffraction_Workflow, however uses an input workspace that is in our system test data directory, and behaves in a similar manner this script.

I'm going to update the script to use the TOPAZ file in our system test directory. I very much doubt that this is going to cause any problems at all, since this is an example script with a hard-coded workspace file. It can't be being used for real data processing.
