Immediate mode updates with simple GUI

Reported by: Owen Arnold Owned by: Michael Reuter
Priority: major Milestone: Iteration 29
Component: VATES Keywords:
Cc: Blocked By:
Blocking: Tester: Owen Arnold

Description

Immediate mode updating for multislice and three slice type views. GUI will immediately be judged against slicomatic, so important to reduce clicks in these type views.

Michael Reuter has raised technical and user-based issues surrounding this functionality. In particular we do not always want to have the Immediate mode updating on, so when do we turn it on and when do we turn it off. In any event, users will expect painted surfaces on planes to update immediately when they are moved, so we should strive to provide this functionality.
